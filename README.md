# CleanShare

**CleanShare** is an Android app that lets you easily rename files and remove metadata (such as EXIF data from photos) before sharing them with others, prioritizing privacy and file hygiene.

---

## Features

- **Rename Files:** Quickly rename documents, images, and other files before sharing.
- **Remove Metadata:** Strip away embedded metadata (like location, device info, authorship, etc.) from photos and media files.
- **Share Securely:** Send files without exposing private or unnecessary information.
- **Simple Interface:** Minimal, user-friendly interface designed for fast workflows.

---

## Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio)
- A device or emulator running Android 7.0 (Nougat) or higher

### Running the App Locally

1. Clone this repository:
    ```bash
    git clone https://github.com/Rockrishna/CleanShare.git
    cd CleanShare
    ```
2. Open the project in **Android Studio** (`File` > `Open` and select the project directory).
3. Let Android Studio synchronize and resolve dependencies.
4. Connect your Android device or start an emulator.
5. Click `Run` or use `Shift + F10` to build and run the app.

---

## How It Works

1. Open the app and select a file you wish to share.
2. Rename the file if desired.
3. Choose to remove metadata (supported file types).
4. Use the app’s sharing feature to send the privacy-safe file.

---

## Why CleanShare?

When you share photos, documents, or other files, hidden metadata (like your device make/model, author name, location, and more) might be included. CleanShare helps you protect your privacy by letting you scrub away this metadata and control the filename before others receive your file.
